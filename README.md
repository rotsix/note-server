## Databases

### users

Stores users' informations (account relative)

- sessions
  - id
  - user_id
  - token
- informations: users information
  - id
  - username
  - password
  - firstname
  - lastname
  - mail
  - birthdate
  - address
  - zip_code
  - city
- configurations: client configurations
